A small system that help you visualize your competitive programing progress

All comands (Typer help):
poetry run clicp --help

Init system:
poetry run clicp init

Delete database:
poetry run clicp reset

Add platform:
poetry run clicp add-platform "<name>"

Add problem:
poetry run clicp add-problem "<name>" --rating "<rating>" --platform "<name>" --contest "<name>"
